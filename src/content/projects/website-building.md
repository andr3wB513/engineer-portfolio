## The approach

No page builders, no bloat: every site is hand-built with
<a href="https://astro.build">Astro</a> as a static site, with self-hosted fonts, a
disciplined color palette, and nothing shipped to the browser that doesn't need to be
there. The result is sites that load instantly and cost almost nothing to run,
deployed to the edge on Cloudflare.
